## 🔷 What Each One Does In One Sentence

**`loader.py`**
Opens a file, reads the text, returns a single string. Nothing more.

**`storage.py`**
Two functions only — `save(chunks_with_vectors)` writes to JSON, `load()` reads it back.

**`ingest.py`**
Calls loader → chunker → embedder → storage in sequence. This is your offline pipeline. You run this **once** before anything else.

**`retriever.py`**
Takes a user query string, embeds it, loops through all stored vectors, runs cosine similarity on each, returns the top K chunk texts.

**`prompt_builder.py`**
Takes a list of chunk texts and a query, returns a formatted prompt string ready to send to an LLM.

**`llm.py`**
Takes a prompt string, calls any LLM API, returns the response text.

**`main.py`**
```
while True:
    query = input("Ask something: ")
    → retrieve
    → build prompt
    → call LLM
    → print answer
```
